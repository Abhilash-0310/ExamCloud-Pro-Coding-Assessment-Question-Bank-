 🚀 ExamCloud Pro – Coding Assessment & Question Bank Management System

ExamCloud Pro is a **full-stack web-based coding assessment and question bank management platform** built using **Java, Spring Boot, Spring MVC, MySQL, Hibernate/JPA, Thymeleaf, HTML, CSS, JavaScript, Maven, and Eclipse IDE**. The platform enables faculty to create coding challenges from platforms like **LeetCode** and **HackerRank**, manage MCQ repositories, assign assessments, and monitor student performance through a centralized dashboard.

---

 📌 Project Overview

ExamCloud Pro is designed for **educational institutions and faculty** to streamline coding assessment management through a centralized and interactive platform. It provides **secure role-based access** for faculty and students, allowing faculty to create and assign coding challenges, import question banks, monitor student completion, and analyze performance efficiently.

The platform combines **coding challenge management, MCQ repositories, assessment scheduling, and student progress tracking** into a single scalable solution.

 🔄 System Workflow

 1️⃣ Faculty Access
Faculty registers or logs into the platform using secure authentication.

 2️⃣ Create Assessments
Faculty creates coding challenges or imports MCQ question banks.

 3️⃣ Add External Questions
Questions can be linked from **LeetCode**, **HackerRank**, or other coding platforms.

 4️⃣ Assign to Students
Faculty assigns coding assessments and question sets to students.

 5️⃣ Student Participation
Students log in and attempt assigned coding challenges.

 6️⃣ Progress Monitoring
Faculty tracks completion status and monitors student performance.

 7️⃣ Analytics Dashboard
Dashboard provides **real-time analytics and assessment insights**.
